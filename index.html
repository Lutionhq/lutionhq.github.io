Equinox

A work-in-progress runtime for Roblox on Linux using containers.

  • Home
  • How To Install
  • GitHub
  • Discord

Equinox is a (sort of new) approach to running Roblox on Linux using containerization. No emulators, no compatibility layers, no GPU passthrough magic. It is fully open source. It uses LXC to run an Android runtime that Roblox comfortably runs inside of, with performance being on-par with Sober*. You can think of it as a native-ish unofficial port of Roblox to Linux. From testing, it is fairly stable despite being experimental software.
