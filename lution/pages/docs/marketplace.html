How to Add Your Mod/Theme to the Lution Marketplace

1. Fork this Repo

Click the Fork button at the top of the page to get started.

2. Add your mod or theme

• Put your mod zip file in: Assets/Mods/Content

• Put your theme zip file in: Assets/Theme/Content

• Drop your thumbnail file in: Assets/thumbnails

3. Drop a pull request

Once you're done, open a pull request and add the Mods/Theme Request label.

You can open one here: Open a pull request

4. What to do next

If you made the mod or just uploaded it to the marketplace, make sure to keep it updated. You can use tools like Killo's modding tool to help with that. (Not sponsored, trust me)
